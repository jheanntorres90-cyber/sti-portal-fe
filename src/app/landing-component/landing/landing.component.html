Operational Workflow

How EduNexus Secures the Campus

Experience the seamless integration of hardware and software designed to keep STI Bacoor students safe and parents informed.

1. Identification & Entry

Students scan their RFID-enabled IDs at the STI Bacoor gate terminal. The system uses facial recognition to verify identity against the student database.

2. Real-time Logging

The local server processes the UID and timestamp, immediately logging the attendance in the MySQL database and updating the Admin Dashboard.

3. Automated Parent Alert

Upon successful verification, the Semaphore SMS Gateway triggers a notification to the registered guardian's mobile number.

4. Student/Admin Insights

Users can log in to the EduNexus web portal to review attendance history, manage profile data, and generate periodic reports.
